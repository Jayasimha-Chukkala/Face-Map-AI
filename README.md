# 🚗 FaceMap AI – Real-Time Driver Drowsiness Detection System

## 📌 Overview

FaceMap AI is an intelligent computer vision application designed to enhance road safety by detecting driver drowsiness in real time. The system leverages **MediaPipe Face Mesh**, **OpenCV**, and **Deep Learning** techniques to analyze facial landmarks from live video streams and identify signs of fatigue before they become dangerous.

By continuously monitoring eye movement patterns, blink behavior, and fatigue-related facial indicators, FaceMap AI helps prevent accidents caused by reduced driver attention and alertness.

---

## ✨ Features

* Real-time facial landmark detection using MediaPipe Face Mesh
* Eye Aspect Ratio (EAR) calculation for blink and eye-closure analysis
* Continuous monitoring of driver alertness levels
* Multi-threshold drowsiness detection mechanism
* Instant audio and visual alerts when fatigue is detected
* Low-latency processing optimized for real-time performance
* Robust performance under varying lighting and driving conditions

## ⚙️ How It Works

1. Captures live video from a webcam.
2. Detects and tracks facial landmarks using MediaPipe Face Mesh.
3. Extracts eye-related features and computes Eye Aspect Ratio (EAR).
4. Monitors blink frequency, eye closure duration, and fatigue indicators.
5. Classifies driver alertness levels using predefined thresholds.
6. Triggers visual and audio alerts when signs of drowsiness are detected.
